# AppFormer.js


Getting started
--
#### 1. Building

##### 1. Core
1. `cd packages/core`
2. `yarn run init && yarn run build`

Build results will appear on `packages/core/dist`

##### 1. ts-exporter
1. `cd packages/ts-exporter`
2. `mvn clean install`

Build results will appear on `packages/ts-exporter/target`
